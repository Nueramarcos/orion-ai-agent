# Contributing to Orion AI Agent

## Development Setup

1. Clone the repository:
   ```sh
   git clone https://github.com/Nueramarcos/orion-ai-agent.git
   cd orion-ai-agent
   ```

2. Set up a virtual environment (optional but recommended):
   ```sh
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install dependencies:
   ```sh
   pip install -e .
   ```

4. Run the tests to ensure everything is set up correctly:
   ```sh
   python -m pytest -q
   ```

## Pull Request Guidelines

1. **Open an Issue**: Before making large changes, open an issue to discuss your proposed changes and get feedback from the maintainers.
2. **Draft PRs Welcome**: Feel free to submit draft pull requests for smaller changes or to start a discussion about larger features.
3. **Follow Coding Standards**: Ensure your code follows the existing coding standards and conventions of the project.
4. **Write Tests**: Include tests with your pull request to ensure that your changes do not break existing functionality.

Thank you for contributing to Orion AI Agent!
